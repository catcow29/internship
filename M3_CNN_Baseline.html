04   Model Configuration
Training Setup
  • Base model: ResNet-50 (ImageNet pretrained), only the final FC layer retrained
  • Optimizer: AdamW (lr=1e-4, weight_decay=1e-2)
  • Scheduler: CosineAnnealingLR (T_max=30)
  • Loss Function: CrossEntropyLoss with class weights to handle imbalance
  • Batch Size: 32 / Epochs: 30 / Early Stopping patience: 5

06   Insights & Reflection
Transfer Learning enabled reaching over 80% performance relatively quickly. However, a gap of approximately 8 percentage points between Train and Test accuracy indicates overfitting tendencies. The next step (M4) will apply Vision Transformer to compare global attention-based performance against the CNN baseline.
