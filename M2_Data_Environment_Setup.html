03   Key Findings
Dataset Structure
  • Dataset used: NIH ChestX-ray14 (subset, approximately 5,000 images)
  • Class composition: 5 classes including Pneumonia, Atelectasis, and Normal
  • Class imbalance confirmed: Normal class accounts for 42% of all samples
Environment Setup
  • Python 3.10 + PyTorch 2.0 + torchvision environment configured
  • Google Colab Pro (GPU: A100) used alongside local development environment
  • Experiment logging configured via Weights & Biases (wandb)
Preprocessing Pipeline
  • Image resizing: all images resized to 224×224 pixels
  • Normalization: ImageNet mean/std applied (mean=[0.485, 0.456, 0.406])
  • Augmentation: RandomHorizontalFlip, RandomRotation(10°), ColorJitter
  • Split ratio: Train 70% / Val 15% / Test 15% with Stratified Split
04   Deliverable
Submitted Jupyter Notebook with data loading, visualization, and preprocessing pipeline (m2_data_setup.ipynb)
05   Insights & Reflection
The class imbalance was more severe than expected. Visualizing the distribution showed that a naive model predicting "Normal" for every sample could still achieve high accuracy — a trivial solution trap. WeightedSampler or Focal Loss will be explored in the next milestone to address this.
